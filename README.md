# playfair

1 - Download repository

2 - cd to /playfair/build

3 - write 'make' to compile


IN ORDER TO RUN THE APPLICATION, WRITE:


./playfair (encode/decode) <keyFilePath> <outputDirectory> <file1> ... <filen>
  
  
./playfair createkey <keyFilePath> <alphabet> <substitute_character> <special_character> <keyword>
